`Select` 在画布上用 **触发条 + 下拉浮层** 展示选项：列表为 **`position: absolute`**（`top: 100%`），**不撑开**上方布局，叠在下方内容上。展开时根节点会设置 **`zIndex`**，且**每次打开**分配递增数值，避免页面上**多个 Select 同时展开**时仍与兄弟同层、被后声明的实例盖住（运行时已支持 `ViewStyle.zIndex`）。依赖指针命中与 **`onClick`**；展开时在下层铺 **全屏遮罩**（相对根容器），点击遮罩或选项外区域可收起（根容器需占满可交互区域时体验最佳，见示例包裹 **`flex: 1`**）。

本版 **不包含** 列表滚动、键盘方向键与读屏语义；长列表请先控制选项数量或待路线图 [滚动能力](/intro/roadmap/) 落地后再扩展。

<Steps>

1. 在应用根部使用 **`CanvasThemeProvider`**（详见 [主题与密度](/ui/theme/)），在 DOM 侧用 **`useCanvasToken()`** 取得 **`token`**。

2. 在 **`<Canvas>`** 子树内渲染 **`Select`** 时须传入 **`token={...}`**（与 [Button](/ui/button/) 相同）。

3. 依赖 **`@lucide/icons`**（`chevron-down`），与包的 peer 一致。

</Steps>

## 参数（Props）

| 属性           | 类型                      | 说明                                                               |
| -------------- | ------------------------- | ------------------------------------------------------------------ |
| `options`      | `SelectOption[]`          | 每项含 **`value`**；**`label`** 可选，未传时用 **`value`** 展示。  |
| `value`        | `string`                  | 受控选中值。                                                       |
| `defaultValue` | `string`                  | 非受控初值。                                                       |
| `onChange`     | `(value: string) => void` | 选中项变化时回调。                                                 |
| `placeholder`  | `ReactNode`               | 可选。未选中或当前值不在 `options` 内时显示；默认 **`"请选择"`**。 |
| `disabled`     | `boolean`                 | 可选。为 true 时不展开。                                           |
| `size`         | `'sm' \| 'md'`            | 可选。默认 **`md`**，影响内边距与字号。                            |
| `token`        | `CanvasToken`             | 在 **`<Canvas>`** 内建议**始终传入**。                             |
| `style`        | `ViewStyle`               | 可选。合并到根容器（默认 **`alignSelf: 'stretch'`**）。            |
| 交互           | —                         | 与 `View` 相同；**`onClick`** 会在触发条点击时调用（切换展开前）。 |

另导出 **`getSelectTriggerShellStyle`**、**`getSelectPanelStyle`**、**`getSelectOptionRowStyle`**、**`selectFontSize`**、**`selectTriggerPadding`** 与 **`SelectSize`**，便于与 **`mergeViewStyles`** 组合或单测。

## 代码示例与预览

与 Button、Loading 等一起的完整主题演示：[@react-canvas/ui Playground](/playground/ui/)。

### 受控与 `placeholder`

```tsx
import { View } from "@react-canvas/react";
import { Select } from "@react-canvas/ui";
import type { CanvasToken } from "@react-canvas/ui";
import { useState } from "react";

function Row({ token }: { token: CanvasToken }) {
  const [value, setValue] = useState<string | undefined>();
  return (
    <View style={{ flex: 1, alignSelf: "stretch" }}>
      <Select
        token={token}
        value={value}
        onChange={setValue}
        placeholder="选一项"
        options={[
          { value: "a", label: "选项 A" },
          { value: "b", label: "选项 B" },
        ]}
      />
    </View>
  );
}
```
