`Button` 是基于 **`View`** 的块级按钮容器，背景/边框来自 **`getButtonStyles(variant, size, token)`**，子节点由你放入 **`Text`**（或其它画布宿主）以控制字号与颜色。

<Steps>

1. 在应用根部使用 **`CanvasThemeProvider`**（详见 [主题与密度](/ui/theme/)），以便在 **DOM 侧** 用 **`useCanvasToken()`** 取 **`token`**。

2. 在 **`<Canvas>`** 子树内渲染 **`Button`** 时，须显式传入 **`token={...}`**（reconciler 子树读不到外层 Context）。

3. 文案请使用 **`<Text style={{ ... }}>…</Text>`** 作为子节点；勿在 **`Button`** 下直接写裸字符串（与运行时规则一致）。

</Steps>

## 参数（Props）

| 属性       | 类型                   | 说明                                                             |
| ---------- | ---------------------- | ---------------------------------------------------------------- |
| `variant`  | `'primary' \| 'ghost'` | 可选。默认 **`primary`**（填充主色）；**`ghost`** 为描边透明底。 |
| `size`     | `'sm' \| 'md'`         | 可选。默认 **`md`**，影响内边距（`paddingSM` / `paddingMD`）。   |
| `disabled` | `boolean`              | 可选。为 true 时 **`opacity: 0.5`** 且不响应交互。               |
| `token`    | `CanvasToken`          | 画布内建议**始终传入**。                                         |
| `style`    | `ViewStyle`            | 可选。与内置样式合并，可覆盖圆角等。                             |
| `children` | `ReactNode`            | 通常为 **`Text`**。                                              |
| 交互       | —                      | 与 `View` 相同（如 **`onClick`**），会透传到外层 `View`。        |

**光标：** 未禁用时 **`Button`** 会合并 **`style.cursor: 'pointer'`**，指针悬停由运行时同步到 **`<canvas>`** 的 DOM **`cursor`**；**`disabled`** 时为 **`'default'`**。需要其它手势时可通过 **`style={{ cursor: '…' }}`** 覆盖（与全局 **`ViewStyle.cursor`** 一致）。

样式辅助函数 **`getButtonStyles`** 与类型 **`ButtonVariant`**、**`ButtonSize`** 由包导出，便于单测或与 **`mergeViewStyles`** 组合。

### `variant` 与 `size`

```tsx
import { Text, View } from "@react-canvas/react";
import { Button } from "@react-canvas/ui";
import type { CanvasToken } from "@react-canvas/ui";

function Toolbar({ token }: { token: CanvasToken }) {
  return (
    <View style={{ flexDirection: "row", flexWrap: "wrap", gap: 12, alignItems: "center" }}>
      <Button token={token} variant="primary" size="md">
        <Text style={{ fontSize: token.fontSizeMD, fontWeight: "600", color: "#ffffff" }}>
          Primary md
        </Text>
      </Button>
      <Button token={token} variant="ghost" size="sm">
        <Text style={{ fontSize: token.fontSizeSM, color: token.colorText }}>Ghost sm</Text>
      </Button>
    </View>
  );
}
```

### `onClick`

画布上使用合成指针事件，常用 **`onClick`** 处理点击（类型见 **`@react-canvas/core`** 的 **`InteractionHandlers`**）。

```tsx
import { Text, View } from "@react-canvas/react";
import { Button } from "@react-canvas/ui";
import type { CanvasToken } from "@react-canvas/ui";
import { useState } from "react";

function CounterButton({ token }: { token: CanvasToken }) {
  const [n, setN] = useState(0);
  return (
    <View style={{ flexDirection: "row", alignItems: "center", gap: 12 }}>
      <Button token={token} variant="primary" size="md" onClick={() => setN((c) => c + 1)}>
        <Text style={{ fontSize: token.fontSizeMD, fontWeight: "600", color: "#ffffff" }}>
          点击
        </Text>
      </Button>
      <Text style={{ fontSize: token.fontSizeSM, color: token.colorText }}>次数：{n}</Text>
    </View>
  );
}
```
