`Avatar` 在圆形区域内展示 **网络图片**、**Lucide 图标数据** 或 **自定义子节点**（常见为 `Text` 首字母）。需安装 **`@lucide/icons`**（与 `@react-canvas/ui` peer 一致）。

<Steps>

1. 使用 `CanvasThemeProvider`（详见 [主题与密度](/ui/theme/)）。

2. 在 `<Canvas>` 内渲染 `Avatar` 并传入 **`token`**。

3. 展示优先级：**`source` 且图片已加载成功** → 图；否则 **`icon`**；否则 **`children`**。图片 **`onError`** 时按同样顺序回退。

</Steps>

## 参数（Props）

| 属性       | 类型                             | 说明                                          |
| ---------- | -------------------------------- | --------------------------------------------- |
| `source`   | `ImageSource`                    | 可选。`{ uri: string }`，加载成功则显示图片。 |
| `icon`     | `LucideIconData`                 | 可选。`@lucide/icons/icons/<name>` 默认导出。 |
| `children` | `ReactNode`                      | 可选。常为 `Text` 单字符。                    |
| `size`     | `number \| 'sm' \| 'md' \| 'lg'` | 可选。默认 **`md`**（由 token 推导像素）。    |
| `token`    | `CanvasToken`                    | 画布内建议显式传入。                          |
| `style`    | `ViewStyle`                      | 可选。合并到外层容器。                        |
| 交互       | —                                | 与 `View` 相同。                              |

## 代码示例与预览

更完整示例见 [@react-canvas/ui Playground](/playground/ui/)。

### `icon` 与 `children`

```tsx
import { Text } from "@react-canvas/react";
import { Avatar } from "@react-canvas/ui";
import user from "@lucide/icons/icons/user";

<Avatar token={token} icon={user} size="md" />
<Avatar token={token} size="md">
  <Text style={{ fontSize: token.fontSizeMD, color: token.colorText, fontWeight: "600" }}>A</Text>
</Avatar>;
```

### `AvatarGroup`

用 **`AvatarGroup`** 包裹多个 **`Avatar`**：横向重叠（`overlap` 控制间距），右侧叠在左侧之上；组内头像带一圈描边（浅色为白边，暗色为 `colorBgLayout`）。

| 属性       | 类型        | 说明                                                  |
| ---------- | ----------- | ----------------------------------------------------- |
| `overlap`  | `number`    | 可选。相邻头像水平重叠量（逻辑像素），默认 **`10`**。 |
| `children` | `ReactNode` | 多个 **`Avatar`**（或其它行内节点）。                 |
| `style`    | `ViewStyle` | 可选。合并到外层行容器。                              |

```tsx
import { Avatar, AvatarGroup } from "@react-canvas/ui";
import user from "@lucide/icons/icons/user";

<AvatarGroup overlap={10}>
  <Avatar token={token} size="md" source={{ uri: "https://example.com/a.jpg" }} icon={user} />
  <Avatar token={token} size="md" source={{ uri: "https://example.com/b.jpg" }} icon={user} />
  <Avatar token={token} size="md" icon={user} />
</AvatarGroup>;
```

### `source`（网络图）

加载过程中可配合 **`icon`** 作为占位；失败时回退到 **`icon`** / **`children`**。

```tsx
import { Avatar } from "@react-canvas/ui";
import user from "@lucide/icons/icons/user";

<Avatar token={token} size="md" source={{ uri: "https://example.com/avatar.jpg" }} icon={user} />;
```
