`Switch` 绘制胶囊轨道与圆形滑块，**`checked`** 控制滑块位置；主题色使用 **`token.colorPrimary`**。

<Steps>

1. 使用 `CanvasThemeProvider`（详见 [主题与密度](/ui/theme/)）。

2. 在 `<Canvas>` 内渲染 `Switch` 并传入 **`token`**。

3. 受控：**`checked`** + **`onChange(checked: boolean)`**；非受控：**`defaultChecked`**。

4. **`disabled`** 时降低透明度且不触发 **`onChange`**。

</Steps>

## 参数（Props）

| 属性             | 类型                         | 说明                                             |
| ---------------- | ---------------------------- | ------------------------------------------------ |
| `checked`        | `boolean`                    | 可选。受控开关状态。                             |
| `defaultChecked` | `boolean`                    | 可选。非受控初值。                               |
| `disabled`       | `boolean`                    | 可选。禁用交互。                                 |
| `size`           | `'sm' \| 'md'`               | 可选。轨道与滑块尺寸，默认 **`md`**。            |
| `onChange`       | `(checked: boolean) => void` | 可选。                                           |
| `token`          | `CanvasToken`                | 画布内建议显式传入。                             |
| `style`          | `ViewStyle`                  | 可选。                                           |
| 交互             | —                            | 与 `View` 相同；内部先切换状态再调用 `onClick`。 |

## 代码示例与预览

完整演示见 [@react-canvas/ui Playground](/playground/ui/)。

### 受控：`md` 与 `sm`

```tsx
import { Text, View } from "@react-canvas/react";
import { Switch } from "@react-canvas/ui";
import type { CanvasToken } from "@react-canvas/ui";
import { useState } from "react";

function Row({ token }: { token: CanvasToken }) {
  const [on, setOn] = useState(true);
  return (
    <View style={{ flexDirection: "row", alignItems: "center", gap: 16 }}>
      <Switch token={token} size="md" checked={on} onChange={setOn} />
      <Text style={{ fontSize: token.fontSizeSM, color: token.colorText }}>{on ? "开" : "关"}</Text>
      <Switch token={token} size="sm" checked={on} onChange={setOn} />
    </View>
  );
}
```
