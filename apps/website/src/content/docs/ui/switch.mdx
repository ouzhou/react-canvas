import { Steps } from "@astrojs/starlight/components";

<Steps>

1. 在应用根部使用 `CanvasThemeProvider`（详见[主题与密度](/ui/theme/)）。

2. 在画布树内渲染 `Switch` 并传入 **`token`**。

3. 受控用法：`checked` + **`onChange(checked: boolean)`**；非受控：`defaultChecked`。可选 **`disabled`** 降低透明度并吞掉点击。

</Steps>

完整示例见 [@react-canvas/ui Playground](/playground/ui/)。
