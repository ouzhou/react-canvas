`Loading` 在画布上展示 **Lucide `loader-circle`** 图标（主题 **`colorPrimary`** 描边），通过 **`requestAnimationFrame`** 驱动 **`style.transform`** 旋转；下方可选说明文案使用 **`colorText`**。尺寸与 [Ant Design Spin](https://ant.design/components/spin-cn) 的指示器规格对齐（约 **14 / 20 / 32** 逻辑像素）。

<Steps>

1. 在应用根部使用 **`CanvasThemeProvider`**（详见 [主题与密度](/ui/theme/)），在 DOM 侧用 **`useCanvasToken()`** 取得 **`token`**。

2. 在 **`<Canvas>`** 子树内渲染 **`Loading`** 时须传入 **`token={...}`**（与 [Button](/ui/button/) 相同，reconciler 子树读不到外层 Context）。

3. 依赖 **`@lucide/icons`**（与 `@react-canvas/ui` 的 peer 一致），用于内置图标数据。

</Steps>

## 参数（Props）

| 属性          | 类型                   | 说明                                                               |
| ------------- | ---------------------- | ------------------------------------------------------------------ |
| `size`        | `'sm' \| 'md' \| 'lg'` | 可选。默认 **`md`**（指示器约 20px；**`sm`** ≈14、**`lg`** ≈32）。 |
| `description` | `ReactNode \| null`    | 可选。未传时为 **`"Loading"`**；传 **`null`** 时不渲染下方文案。   |
| `spinning`    | `boolean`              | 可选。默认 **`true`**；为 **`false`** 时图标不旋转。               |
| `token`       | `CanvasToken`          | 在 **`<Canvas>`** 内建议**始终传入**。                             |
| `style`       | `ViewStyle`            | 可选。合并到根容器（纵向居中排列图标与文案）。                     |

另导出 **`getLoadingMetrics(size)`** 与 **`LoadingSize`**，便于单测或与 **`mergeViewStyles`** 组合。

## 代码示例与预览

与 Checkbox、Button 等一起的完整主题演示：[@react-canvas/ui Playground](/playground/ui/)。

### `size`

```tsx
import { View } from "@react-canvas/react";
import { Loading } from "@react-canvas/ui";
import type { CanvasToken } from "@react-canvas/ui";

function Row({ token }: { token: CanvasToken }) {
  return (
    <View style={{ flexDirection: "row", gap: 24, alignItems: "flex-end" }}>
      <Loading token={token} size="sm" description="小号" />
      <Loading token={token} size="md" description="默认" />
      <Loading token={token} size="lg" description="大号" />
    </View>
  );
}
```
