`Divider` 在画布内绘制 **1px** 分隔线，颜色取自主题 **`token.colorBorder`**。无子节点时为单线；有子节点时在中间渲染文案，两侧为线段（横排 **左 | 文 | 右**，竖排 **上 | 文 | 下**）。

<Aside type="note">
  与 [Button](/ui/button/) 相同：在 **`<Canvas>`** 子树中须传入 **`token={...}`**（由外层 `useCanvasToken()` 取得），无法依赖外层 `CanvasThemeProvider` 的 React Context。
</Aside>

## 用法

```tsx
import { Canvas, CanvasProvider, Text, View } from "@react-canvas/react";
import { CanvasThemeProvider, Divider, useCanvasToken } from "@react-canvas/ui";

function Example() {
  const token = useCanvasToken();
  return (
    <CanvasProvider>
      {({ isReady, error }) =>
        isReady && !error ? (
          <Canvas width={320} height={120}>
            <View style={{ flex: 1, padding: 16, backgroundColor: token.colorBgLayout }}>
              <Divider token={token} orientation="horizontal" />
              <Divider token={token} orientation="horizontal">
                <Text style={{ fontSize: token.fontSizeSM, color: token.colorText }}>标签</Text>
              </Divider>
            </View>
          </Canvas>
        ) : null
      }
    </CanvasProvider>
  );
}

// 外层（DOM 侧）包裹主题，供 useCanvasToken 使用
export function App() {
  return (
    <CanvasThemeProvider theme={{ appearance: "light", density: "default" }}>
      <Example />
    </CanvasThemeProvider>
  );
}
```

竖线置于 **横向** `View`（`flexDirection: "row"`）中时，通常设置 **`style={{ alignSelf: "stretch" }}`**，使线在交叉轴拉满高度。

样式辅助函数（与 `getButtonStyles` 并列）：

- **`getDividerLineStyle(orientation, token)`**：纯线单条样式。
- **`getDividerSegmentStyle(orientation, token)`**：带文案时两侧线段样式（内部使用，也可在外层自定义合并）。

## 参数（Props）

| 属性          | 类型                         | 说明                                                                                                             |
| ------------- | ---------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `orientation` | `'horizontal' \| 'vertical'` | 可选，默认 **`horizontal`**。纯线：横线高度为 1、竖线宽度为 1。有子节点：横为主轴 **row**，竖为主轴 **column**。 |
| `token`       | `CanvasToken`                | 在 `<Canvas>` 内 **须传入**。                                                                                    |
| `style`       | `ViewStyle`                  | 可选。合并到根 `View`（纯线或带文案容器）。                                                                      |
| `children`    | `ReactNode`                  | 可选。有则渲染带中间文案变体；中间请使用 **`<Text>`** 等画布节点（包内不对字符串自动包 `Text`）。                |
| 交互          | `InteractionHandlers`        | 可选。与 `View` 一致，透传根节点。                                                                               |
