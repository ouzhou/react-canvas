## 摘要

**阶段二**：在 `View` 下使用 **`Text`**，由 Skia Paragraph 测量与绘制；支持嵌套 `Text`、换行、`numberOfLines` 等。结构约束见 [运行时结构约束](/reference/constraints/)（如 **`Text` 内不可嵌套 `View`**）。

<Tabs>
  <TabItem label="Core（TS）">
    场景侧使用 **`TextNode`** 与段落构建 API（如 **`collectParagraphSpans`**、**`paintNode`** 路径中的文字分支）。字体注册与默认字体见包内 **`ensureDefaultParagraphFonts`** 等导出。

    ```ts
    import { TextNode, ViewNode } from "@react-canvas/core";

    // 概念：TextNode 作为子节点挂到 View 下，由布局与绘制管线消费
    ```

    细节以源码与 [react-yoga-text-nodes.md](https://github.com/ouzhou/react-canvas/blob/main/docs/react/react-yoga-text-nodes.md) 为准。

  </TabItem>
  <TabItem label="React">
    ```tsx
    import { CanvasProvider, Canvas, View, Text } from "@react-canvas/react";

    <CanvasProvider>
      <Canvas width={300} height={200}>
        <View style={{ padding: 8 }}>
          <Text style={{ fontSize: 16 }}>Hello</Text>
        </View>
      </Canvas>
    </CanvasProvider>
    ```

  </TabItem>
</Tabs>
