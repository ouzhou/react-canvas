## 摘要

**阶段四（多媒体）**：**`Image`** 异步解码与缓存、**`SvgPath`** 矢量路径绘制等。滚动与 `overflow` 裁剪为后续 Step（见 [路线图](https://github.com/ouzhou/react-canvas/blob/main/docs/development-roadmap.md)）。

<Tabs>
  <TabItem label="Core（TS）">
    使用 **`ImageNode`**、**`SvgPathNode`** 与图像解码/缓存工具（如 **`decodeImageFromEncoded`**、**`computeImageSrcDestRects`**）。绘制由统一 **`paintNode`** 管线处理。

    ```ts
    import { ImageNode, SvgPathNode } from "@react-canvas/core";
    ```

  </TabItem>
  <TabItem label="React">
    ```tsx
    import { Image, SvgPath } from "@react-canvas/react";

    <Image source={{ uri: "..." }} style={{ width: 120, height: 80 }} />
    <SvgPath d="M0 0 L100 100" style={{ strokeWidth: 2 }} />
    ```

  </TabItem>
</Tabs>
