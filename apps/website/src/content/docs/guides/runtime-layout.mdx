## 摘要

对应 **阶段一** 与里程碑 **M1**：并行初始化 **Yoga** 与 **CanvasKit**，用 **Flexbox** 计算布局，再用 Skia 将场景树绘制到 Surface。完整设计见仓库专文。

<Tabs>
  <TabItem label="Core（TS）">
    使用 **`initCanvasRuntime`** 得到 **`CanvasRuntime`**，构造 **`ViewNode`** 树，调用 **`calculateLayoutRoot`** / **`syncLayoutFromYoga`**，再 **`paintScene`** 输出到 Surface。

    ```ts
    import {
      initCanvasRuntime,
      ViewNode,
      calculateLayoutRoot,
      paintScene,
    } from "@react-canvas/core";

    const runtime = await initCanvasRuntime();
    const root = new ViewNode(runtime.yoga, "View");
    root.setStyle({ width: 300, height: 200 });
    calculateLayoutRoot(root, 300, 200, runtime.yoga.DIRECTION_LTR, runtime.canvasKit);
    // paintScene(root, skCanvas, runtime.canvasKit, dpr, paint);
    ```

    绘制需有效的 Skia **`Canvas`** 与 **`Paint`**；具体见包内类型与 Playground。

  </TabItem>
  <TabItem label="React">
    使用 **`CanvasProvider`** 提供运行时，在 **`Canvas`** 下挂载 **`View`**，用 `style` 表达 Flexbox（对齐 React Native 语义）。

    ```tsx
    import { CanvasProvider, Canvas, View } from "@react-canvas/react";

    export function App() {
      return (
        <CanvasProvider>
          <Canvas width={300} height={200}>
            <View style={{ flex: 1, backgroundColor: "#3366cc" }} />
          </Canvas>
        </CanvasProvider>
      );
    }
    ```

    **`Canvas` 的直接子节点只能是单个 `View`**（当前约束）；详见 [运行时结构约束](/reference/constraints/)。

  </TabItem>
</Tabs>
